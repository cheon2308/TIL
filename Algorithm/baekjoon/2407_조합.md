[백준 2407_조합](https://www.acmicpc.net/problem/2407)

## 조건

- nCm 을 구하여라


## 접근 방법

![](assets/Pasted%20image%2020221103175837.png)


- 입력이 크지 않기 때문에 dp 이용해주지 않아도 될 것 같다.

```python
def fac(n):
    num = 1
    for i in range(2,n+1):
        num*=i
    return num

n,m = map(int,input().split())
print(fac(n) // (fac(m)*fac(n-m)))
```
