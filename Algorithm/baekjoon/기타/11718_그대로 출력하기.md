[백준 11718](https://www.acmicpc.net/problem/11718)

## 그대로 출력하기

```python
# 오답
while True:
    print(input())
```

###### 결과 -> EOFError

- 몇 개의 입력이 주어지는지 나와 있지 않아 입력값이 들어오지 않는 경우 에러가 발생하였습니다.

```python
# 정답
while True:
    try:
        print(input())
    except EOFError:
        break
```

###### 해결방법 -> Try, Except

- 예외 구문을 이용하여 입력이 없는 경우 break을 통해 반복문 탈출을 하였습니다.
